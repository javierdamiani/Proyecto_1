## 🚀 Instalación y Desarrollo

### Prerrequisitos
- Node.js (para SASS)
- Navegador moderno

### Compilación de SASS
```bash
# Instalar SASS globalmente
npm install -g sass

# Compilar SCSS a CSS
sass scss/main.scss css/style.css --style=expanded

# Compilar versión minificada
sass scss/main.scss css/style.css --style=compressed
```

### Estructura SCSS
- `_variables.scss`: Variables de colores, espaciado, tipografía
- `_mixins.scss`: Mixins reutilizables y media queries
- `_placeholders.scss`: Placeholders para componentes comunes
- `_animations.scss`: Keyframes y clases de animación
- `_components.scss`: Estilos de componentes específicos
- `_base.scss`: Estilos base y reset
- `main.scss`: Archivo principal que importa todos los módulos
